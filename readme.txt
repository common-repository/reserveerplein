Plaats eenvoudig de ReserveerPlein plugin op uw website.

1. Installeer deze plugin
2. Kopieer de Shortcode uit uw ReserveerPlein account
3. Plaats deze shortcode op de door u gewenste plek
4. Uw bezoekers kunnen reserveren middels de widget!
